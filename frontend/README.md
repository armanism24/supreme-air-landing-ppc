# Supreme Air Austin Website

Professional landing page for Austin's #1 air duct cleaning company.

## Quick Start

```bash
# Install dependencies
yarn install

# Start development server
yarn start
```

The website will open at `http://localhost:3000`

## Features

✅ **Auto-play Video** - Professional air duct cleaning demonstration  
✅ **Lead Capture Forms** - EmailJS integration for direct email delivery  
✅ **Mobile Responsive** - Optimized for all devices  
✅ **Professional Design** - Conversion-optimized with Supreme Air branding  
✅ **Before/After Gallery** - Real customer results  
✅ **Health Facts Section** - EPA/CDC data on air quality  
✅ **Customer Reviews** - 768+ Google reviews display  

## EmailJS Setup

To activate contact form emails, follow the setup guide in `EMAILJS_SETUP.md`

## Technologies Used

- React 19
- Tailwind CSS
- Shadcn/UI Components  
- EmailJS (for contact forms)
- Lucide React (icons)
